# iran-conflict-dashboard

Minimalist dashboard using real, cited open-source data snapshots for the Iran/UAE escalation context.

## Data sources
- Al Jazeera news feed/headlines: https://www.aljazeera.com/news/
- Dubai Airports advisory: https://www.dubaiairports.ae/
- UAE Ministry of Foreign Affairs: https://www.mofa.gov.ae/en

## Notes
- This is a snapshot dashboard (manually updated from verified source pulls).
- It is designed for quick situational awareness, not official operational control.
